# iAbroad – 留学生一站式生活服务平台  
**iAbroad – One-Stop Life Service Platform for International Students**

> *织密留学生活服务网，点亮异国他乡安居路*  
> *Weaving a Dense Service Network for International Students, Lighting Up the Path to a Safe and Comfortable Life Abroad*

---

## 📖 Introduction | 项目简介

**iAbroad** is a WeChat-based mini-program designed specifically for international students in China. It integrates **rental services, group buying, second-hand marketplace, travel recommendations, and community forums** into one platform — solving language barriers, information asymmetry, and high living costs.

**iAbroad** 是一款专为来华留学生打造的微信小程序，整合 **租房、团购、二手交易、旅游推荐、社区讨论** 五大核心功能，一站式解决留学生在华生活中的语言障碍、信息不对称、生活成本高等痛点。

---

## 🎯 Mission & Vision | 使命与愿景

**Mission**  
Break information barriers, reduce living costs, enhance campus life quality, and promote cross-cultural communication among international students.

**Vision**  
Become the most trusted life companion for international students in China, starting from Jiangsu Normal University and expanding nationwide.

**使命**  
打破信息壁垒，降低生活成本，提升校园生活品质，促进中外学生文化交流。

**愿景**  
从江苏师范大学出发，逐步覆盖全国高校，成为来华留学生最信赖的生活伙伴。

## ✨ Key Features | 核心功能

| Feature | Description |
|---------|-------------|
| 🏠 **Rental Service** | Bilingual listings, online booking, contract guidance, roommate matching |
| 🛒 **Group Buying** | Exclusive discounts on food, groceries, and daily necessities |
| 📦 **Second-Hand Market** | Buy/sell textbooks, electronics, furniture — with payment guarantee |
| 🗺️ **Travel Recommendations** | Itinerary planning, safety tips, travel insurance, destination guides |
| 💬 **Community Forum** | Multi-language support, Q&A, event organizing, cultural exchange |

| 功能 | 描述 |
|------|------|
| 🏠 **租房服务** | 双语房源、在线预约、签约指引、合租匹配 |
| 🛒 **团购专区** | 餐饮、超市等专属优惠，省钱省心 |
| 📦 **二手市场** | 教材、电器、家具买卖 + 第三方交易担保 |
| 🗺️ **旅游推荐** | 行程规划、安全提醒、保险购买、目的地攻略 |
| 💬 **社区讨论** | 多语言交流、求助问答、线下活动组织 |

---

## 🧠 Innovativeness | 创新亮点

- **Service Integration** – First platform to merge housing, shopping, second-hand trading, travel, and social features for international students.
- **Multi-language Support** – Real-time translation API covering UI, posts, and chats.
- **Blockchain-backed Trust** – Tamper-proof transaction records for rentals and second-hand trades.
- **Cultural Fusion** – Online + offline events (cultural festivals, language corners, food fairs).

- **服务融合** – 首次为留学生整合租房、团购、二手、旅游、社交五大场景。
- **多语言支持** – 集成实时翻译API，覆盖界面、帖子、聊天全场景。
- **区块链存证** – 租房及二手交易记录不可篡改，提升信任。
- **文化融合** – 线上线下结合的文化节、语言角、美食分享会等活动。
